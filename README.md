# 🕵️‍♂️ News Fact-Checker (ระบบวิเคราะห์ความน่าเชื่อถือของข่าวออนไลน์)

ระบบตรวจสอบความน่าเชื่อถือของข่าวและเนื้อหาบนโลกออนไลน์ โครงงานนี้ผสานการทำงานของการสกัดเนื้อหาเว็บ (Web Scraping), การสืบค้นฐานข้อมูลข่าว (Search Engine) และการใช้ Large Language Model (LLM) ในการประเมินความน่าเชื่อถือ

## 📦 โครงสร้างไฟล์หลัก
- `app.py`: ไฟล์หลักสำหรับรันหน้าเว็บแอปพลิเคชัน (Streamlit)
- `search.py`: ระบบสืบค้นและเปรียบเทียบข้อมูลจาก Google News RSS
- `scraper.py`: ระบบสกัดข้อความจาก URL และ Social Media
- `llm.py`: ระบบเชื่อมต่อกับ Qwen-2.5 (7B) เพื่อวิเคราะห์ความน่าเชื่อถือ
- `requirements.txt`: รายชื่อไลบรารีที่จำเป็นสำหรับโปรเจคนี้

## 🚀 วิธีการติดตั้งและใช้งาน
1. ติดตั้งไลบรารี: `pip install -r requirements.txt`
2. สร้างไฟล์ `.env` และใส่ค่า `OPENROUTER_API_KEY`
3. รันระบบ: `streamlit run app.py`
